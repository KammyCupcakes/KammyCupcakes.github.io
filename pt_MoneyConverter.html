I made this page back in 2021 during my spring semester at BHCC(My Junior Year). This page shows a money converter I made using mainly javascript function loops to display it to the user. I made this page for my college JavaScript course I was taking at Bunker Hill. We had just learned how to use functions 2 weeks before this and I was still get used to it because it had me a bit lost with how it worked. This assignment allowed me to understand how to set up the function and the for loops properly and having them switch for his type of currency using different functions for each. Then implementing buttons at the end of the code so there is a "onClick" command to display it to the user. To display everything on the screen we also used document.write to get it up on the page. Mainly everything was handled through the JavaScript.

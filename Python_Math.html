while loops With the while loop can execute a set of statements as long as a condition is true. k = 1
while k < 9:
print(k)
k += 1
K would print number 1 to 8
break Statement The break statement can stop the loop even if the while condition is true k = 1
while k < 9:
print(k)
if (k == 6):
break
k += 1
K would print up until k is equal to 6
continue Statement The continue statement can stop the current iteration, and continue wih the next. k = 1
while k < 9:
print(k)
if k == 6:
continue
print(k)
K would print every number up to 9 except 6.
else Statement The else statement allows us to run a block of code when the condition no longer is true. k = 1
while k < 9:
print(k)
k += 1
else:
print("k is no longer less than 9")
Once the condition is false (when it reaches 8) the print statement will be displayed.

Python For Loops

Name Definition Example
for Loop for loops are used when you have a block of code which you want to repeat a fixed number of times. The for-loop is always used in combination with an iterable object, like a list or a range. fruits = ["apple", "banana", "cherry"]
for x in fruits:
print(x)
Looping Through a String Similarly, strings are iterable objects as they contain a sequence of character. for x in "kameron":
print(x)
This will print out each letter in the name "kameron"
break Statement The break statement can stop a loop before it has looped through all the items. games = ["Valorant", "Counter-Strike", "Rainbow Six Siege"]
for x in games:
print(x) if x == "Counter-Strike":
break
 The break statement at the end would make the loop stop once it gets to "Counter-Strike"
continue Statement The continue statement we can stop the current iteration of the loop and continue with the proceeding one. games = ["Valorant", "Counter-Strike", "Rainbow Six Siege"]
for x in games:
if x == "Counter-Strike":
continue print(x)
 This code would display "Valorant" and "Rainbow Six Siege" since the continue is set to skip "Counter-Strike"
The range() Function The range() function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number. for x in range(6):
print(x)
 This displays numbers in the range of 0 to 5.
for x in range of (3, 9):
print(x)
 This displays numbers from 3 to 8.
for x in range(2, 50, 3):
print(x)
 This will display numbers from 2 to 47, but due to adding that third parameter (3) it will now go up by an increment of 3.
else The else keyword in a for loop specifies a block of code to be executed when the loop is finished. for x in range(6):
print(x)
else:
print("Finally finished")
 This would display numbers from 0 to 5 and once 5 is reached the print statement is displayed to the user letting them know the for loop condition is false
pass Statement If your loop was empty and had no content, put in the pass statement to avoid getting an error. for x in [0, 1, 2]
pass
 This code doesn't display anything but if the pass statement wasn't there it would raise an error.
Nested loops A nested loop is a loop inside a loop.
The "inner loop" will be executed/run one time for each iteration of the "outer loop" 		adj = ["", "", ""]

Condition Statements Condition statements are used to evaluate the code to see if it meets the specified conditions.
  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b
if Statements if statements allow for conditional execution of a statement or group of statements based on the value of an expression. k = 18
j = 20
if j > k:
print("j is greater than k")
 This code compares the values assigned to j & k, so if j is greater than k it will print "j is greater than k".
Indentation Python relies on indentation (whitespace at the beginning of a line) to define scope in the code. If statement, without proper indentation:
j = 20
k = 18
if j > k
print("j is greater than k")
 This would give an error since the print statement is not indented to be inside of the if statement. (This also matters for elif statements & else statements.)
elif Statements The elif statement is pythons way of saying "if the previous conditions were not true, then try this condition" j = 20
k = 20
if j > k:
print("j is greater than k")
elif j == k:
print("j and k are equal")
 This code would display the print statement from the elif as the condition was true and the ifs condition was false

Name Tag/Element Definition
Form form | Attributes: action The form tag is used to create an HTML form for the user input.
Input input | Attributes: name, button, date, email, number, password, radio, text, submit, type, id The input tag specifies an input field where the user can enter data and can be displayed in several ways, depending on the type attribute.
  • radio: Displays a radio button where the user can select one choice
  • checkbox: Displays a checkbox where the user can select multiple choices
  • text: Displays a single-line text input field
Label label | Attributes: for The label tag defines a label for input type elements:
  • input type="checkbox"
  • input type="name"
  • input type="password"
  • input type="text"
  • input type="date"
Button button | Attributes: type, name The button tag defines a clickable button.
Fieldset fieldset | Attributes: name The fieldset tag is used to group related elements in a form. It also creates/draws a box around those elements. The legend tag is used to define a caption for the fieldset element
Option option | Attributes: value The option tag defines an option in a select list. option elements go inside other elements like select, optgroup, datalist.
Textarea textarea | Atrributes: rows, cols, name, id. The textarea tag defines a multi-line textbox. This element is used in forms to collect user inputs like reviews or additional comments

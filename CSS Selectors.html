Name

Tag/Element

Definition
Class Selector .class The .class selector selects all elements with a specific class attribute/name.
ID Selector #id The #id selector styles the element with the specified id.
Element Selector p The element selector selects all elements with the specified element name.
Element Class Selector p The element.class selector is used to select the specified element with the specified class.
Element, Element Selector h1, h2 Allows for two or more elements to be selected and styled at once by using a comma.
Element Element Selector tr td The element element selector is used to select elements inside elements.
Element>Element Selector tr > td The element>element selector is used to select elements with a specific parent.
Element+Element Selector tr + td The element+element selector is used to select an element that Is directly after another specific element.
Element~Element Selector h1 ~ ul The element1~element2 selector matches occurrences of element2 that are preceded by element1. Both elements must have the same parent, but element2 does not have to be immediately preceded by element1.
* Selector * or div * The * selector selects all elements. The * selector can also select all elements inside another element
:active Selector a:active The :active selector is used to select and style the active link. Once you click a link it will become active. :active MUST come after :hover to work.
::after Selector p::after The ::after selector insert something after the content of each selected element(s).
::before Selector p::before The ::before selector inserts something before the content of each selected element(s).
:link Selector a:link The :link selector is used to select unvisted links.
:visted Selector a:visited The :visited selector is used to select visited links.
:hover Selector a:hover The :hover selector is used to select elements when you hover over them.
:target Selector div:target The :target selector can be used on URLs with an # followed by an anchor name link to a certain element.
:first-child Selector p:first-child The :first-child selector is used to select the specified selector, only if it is the first child of its parent.
:only-child Selector p:only-child The :only-child selector matches every element that is the only child of its parent.
:last-child Selector p:last-child The :last-child selector matches every element taht is the only child of its parent.
::first-letter Selector p::first-letter The ::first-letter selector is used to add a style to the first letter of the specified selector. It can only be used with block-level elements.

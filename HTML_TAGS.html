Name Tag/Element Definition
Declaration !DOCTYPE All HTML documents must start with a !DOCTYPE declaration. It is an 'information' to the browser about what document type to expect.
HTML HTML (Required) The html tag is a required HTML tag that you would put at the beginning and end of your HTML file
Head head (Required) The head holds information like the title, meta tags, content type, and links to CSS stylesheets and JavaScript
Title title (Required) This is the text that goes in the title bar or the browser window.
Body body (Required) This contains the contents of the program.
Paragraph p This tag allows you to create lines of text or in this case "paragraphs."
Headings h1, h2, h3, h4, h5, h6 The heading tags (h1 - h6) are typically used for the title of a page or post and it is the first header visible on a page.
Horizontal Line hr This is a horizontal line. You can use width and size attributes can be applied to the tag/line.
Break br This tag allows you to insert line breaks
